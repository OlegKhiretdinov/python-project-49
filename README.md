«Игры разума» — набор из пяти консольных игр

### Минимальные требования

Python 3.10
Pip 23
Poetry 1.4

### Установка

1. Склонировать репозиторий
2. В корневой папке выполнить команды  
   `make install`  
   `make build`  
   `make package-install`
